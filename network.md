**宿主机上先启动一个临时监听服务**

meteor@DESKTOP-E75SD7O:~/bar$ python3 -m http.server 8080 --bind 10.0.0.1
Serving HTTP on 10.0.0.1 port 8080 (http://10.0.0.1:8080/) ...
10.0.0.2 - - [29/Apr/2026 11:36:29] "GET / HTTP/1.1" 200 -


容器里：

启动：
```
  sudo ./bin/barco \
  -u 0 \
  -m / \
  -c /bin/sh \
  --veth eth0 \
  --container-ip 10.0.0.2/24 \
  --host-ip 10.0.0.1/24 \
  --nat \
  --dns 8.8.8.8
```
